## Run Locally

Clone the project

```bash
  git clone https://github.com/pranav043/test-harmoney.git
```

Go to the project directory

```bash
  cd test-harmoney
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Open the browser & visit link displayed on console
